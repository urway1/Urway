# تيشرت URWAY

موقع بسيط لعرض تيشرت مطبوع للبيع.  
يعرض معلومات التيشرت، السعر، والمقاسات.  
يتوفر زر لإرسال الطلب عبر واتساب.

## التفاصيل

- السعر: 150 د.م (بعد التخفيض من 199 د.م)  
- المقاسات: S, M, L  
- قماش عالي الجودة  
- توصيل مجاني لجميع المدن
